## Copy Guidelines

### Word Limits

| Element | Max Words |
|---------|-----------|
| Headline | 8 |
| Subheadline | 15 |
| Section title | 6 |
| Feature title | 4 |
| Feature description | 12 |
| Testimonial | 10 |
| CTA | 3 |

### Tone

- Direct
- Confident
- Outcome-focused
- No fluff

### Keywords (use sparingly)

- Earn / Earning / Income
- Hyper-realistic
- Consistent / Consistency
- One-click
- 24/7

### Avoid

- "Revolutionary", "Game-changing", "Cutting-edge"
- Long explanations
- Technical jargon
- Multiple CTAs per section
