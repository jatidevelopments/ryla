## Overview

This document lists all available model types and categories that can be downloaded through the ComfyUI Manager interface on our RunPod server. The list is organized by model type categories and base model architectures.

> **Note**: This list contains model **categories/types**, not individual model files. To see the full list of downloadable models, you need to browse the Model Manager interface directly. The actual model list is extensive and dynamically loaded.

## How to Access the Full Model List

1. Navigate to the ComfyUI interface: https://5qj51a8nptsc4h-8188.proxy.runpod.net/
2. Click the **⚙️** (Settings) button in the top right
3. Click **ComfyUI Manager** from the menu
4. Click **Model Manager** button
5. Use the filters to browse:
   - **Filter**: All / Installed / Not Installed / In Workflow
   - **Type**: Select from the model type categories above
   - **Base**: Select from the base architectures above
   - **Search**: Search for specific model names

## Notes

- The model list is dynamically loaded and may change as new models are added to the ComfyUI ecosystem
- Individual model files are not listed here - only the categories/types
- To see all available individual models, browse the Model Manager interface directly
- Models can be filtered by installation status, type, and base architecture
- Each model entry shows an "Install" button if not already installed
