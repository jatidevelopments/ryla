## Overview

ZenCreator is an AI-powered content creation platform offering a comprehensive suite of tools for image and video generation, editing, and manipulation. The platform focuses on influencer-style content creation with specialized tools for face generation, body modification, and video production.

## Available Tools

### 1. Image-To-Video
- **Description:** Converts static images into animated videos
- **Use Case:** Creating dynamic content from photos
- **URL:** `/tools/video-generator`
- **Models Available:**
  - **Kling 1.6 - Start Frame** (Fast, basic motion, quick results)
  - **Kling 2.1 - Start/End Frame** (Sharper, smoother, more realistic)
  - **Kling 2.5 - Start Frame** (Latest version with enhanced quality)
  - **WAN - Start/End Frame** (Full freedom, uncensored)
  - **WAN + Audio - Start Frame & Audio File** (Audio-driven generation)
  - **Seedance Pro Fast - Start Frame** (default)
  - **Seedance Pro - Start/End Frame**

#### Image-To-Video Features:
- **Start Frame Upload:** Upload image(s) to generate video from (up to 100 files)
- **End Frame:** Optional end frame for some models (Kling 2.1, WAN, Seedance Pro)
- **Audio File:** Optional audio file for WAN + Audio model
- **Prompt:** Optional text description of desired video
- **Resolution:** Configurable (default: 720p, dropdown suggests other options available)
- **Duration:** Configurable (default: 5 seconds, dropdown suggests other options available)
- **Template Library:** 
  - Pre-built video templates available
  - "Task" and "Library" tabs for template selection
  - Templates include video previews with play buttons
  - Each template has a "Try Template" button and "Copy ID" functionality
- **Credit Cost:** 4 credits per video
- **Watermark:** First 30 credits generate with ZenCreator watermark

### 2. Video-To-Video

- **URL:** `/tools/video-to-video`
- **Model Available:**

#### Video-To-Video Features:
- **Image Upload:** Upload image to apply style/appearance to video (single file)

- **Resolution:** Configurable (default: 480p, dropdown suggests other options available)

### Potential Weaknesses
