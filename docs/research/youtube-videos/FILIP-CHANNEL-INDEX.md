## Key Themes Across All Videos

### 1. Quality vs Speed Trade-off
- API models (90% quality) vs ComfyUI+LoRAs (100% quality)
- Cost considerations: $0.03/image vs free after training
- Learning curve: Months for ComfyUI vs instant for APIs

### 2. Automation is Critical
- Batch generation essential for scale
- Airtable/workflow automation saves time
- Script-based downloads and processing

### 3. Marketing > Quality
- "Best pictures + bad marketing = bad results"
- "Okay pictures + good marketing = good results"
- Focus on reach and conversion, not perfection

### 4. Consistency Methods
- LoRA training = best method (100% consistency)
- Image edit models = quick method (80-90% consistency)
- Face swap = fallback (variable quality)

### 5. Realism Techniques
- Prompt engineering (amateur photo, cell phone quality)
- Post-processing (Photoshop, grain, shadows)
- Realism LoRAs (smartphone snapshot photo reality)
- Avoid: 8K, 4K, ultra realistic prompts
