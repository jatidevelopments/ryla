## Summary

Tutorial on creating talking/lip-synced videos of AI influencers. Covers complete workflow: image generation (close-up for lip sync), voice generation (11 Labs), and lip sync (Kling Infinite Talk or Clink Avatars 2.0).

Workflow:
1. Generate close-up image (NanoBanana Pro via Gemini)
2. Create voice (11 Labs text-to-speech)
3. Upload image + audio to lip sync API (fal.ai, Wavespeed, or Hicksfield)
4. Generate talking video
5. Optional: Video-to-video refinement (Clink Lip Sync)

## Key Points

- **API Models Preferred**: Faster than local ComfyUI (30 min vs 5-10 min)
- **Close-Up Required**: Image must be head/chest only for best results
- **Voice First**: Generate voice, then lip sync
- **Multiple Attempts**: May need to regenerate for best quality
- **Short Videos**: 10-second clips better than 60-second
