## Generation Instructions

1. Use a high-quality image generation service (Replicate, Midjourney, DALL-E, or Stable Diffusion)
2. Generate each image with the provided prompt
3. Optimize images to WebP format
4. Resize to 480x640px (maintain aspect ratio, crop if needed)
5. Compress to < 200KB per image
6. Save to `apps/web/public/` directory
7. Name files exactly as specified: `auth-promo-1.webp` through `auth-promo-5.webp`

## Usage

The auth page will automatically cycle through these images on desktop view. The images should showcase the diversity and quality of AI influencers that can be created with RYLA.
